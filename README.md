# 💬 Creatix Chat

تطبيق **دردشة (Chat)** لأجهزة أندرويد، مبني بـ **Kotlin** و **Jetpack Compose**، ومربوط بالكامل مع **Firebase** للمصادقة، التخزين، والإشعارات الفورية.

---

## ✨ المميزات

- 🔐 **تسجيل دخول وتسجيل حساب** عن طريق Firebase Authentication
- 💾 **تخزين سحابي فوري** للمحادثات باستخدام Cloud Firestore
- 🔔 **إشعارات Push** لحظة وصول أي رسالة جديدة
- 🎨 **واجهة حديثة** مبنية بالكامل بـ Jetpack Compose و Material 3
- 🖼️ **دعم الصور الشخصية (Avatar)** عبر مكتبة Coil
- 📱 **يدعم من Android 8.0 (API 26)** وحتى أحدث إصدار

---

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---|---|
| *Kotlin* | لغة البرمجة الأساسية |
| *Jetpack Compose* | بناء الواجهات |
| *Firebase Auth* | تسجيل الدخول |
| *Cloud Firestore* | قاعدة البيانات الفورية |
| *Firebase Messaging* | الإشعارات |
| *Coil* | عرض الصور |
| *OkHttp* | التعامل مع الشبكة |
